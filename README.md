# javascript-lotto-precourse

## 진행 순서도

- [ ] 로또 구입 입력문 출력 및 사용자 입력
- [ ] 입력값 유효성 검사

  - [ ] 양의 정수인가
  - [ ] 문자가 포함되었는가
  - [ ] 1000원 단위인가

- [ ] 로또 구매 개수 출력
- [ ] 구매 개수만큼 로또 번호 출력

  - [ ] 로또 번호가 오름차순인가
  - [ ] 미션 유틸로 6개의 숫자를 반환했는가

- [ ] 당첨 번호 입력문 출력 및 사용자 입력
- [ ] 입력 번호들 유효성 검사
  - [ ] 6개의 숫자가 중복되지 않는가
  - [ ] 특수문자가 포함되었는가
  - [ ] 로또 번호 범위에 포함되는가
- [ ] 보너스 번호 입력문 출력
- [ ] 입력 번호 유효성 검사
- [ ] 로또별 당첨 개수 측정
- [ ] 당첨 결과 출력
  - [ ] 3개, 4개, 5개, 5개+보너스, 6개 일치 결과
  - [ ] 수익률 출력
    - [ ] 소수점 둘째 자리에서 반올림 여부
