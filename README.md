# javascript-lotto-precourse

#### 로또를 구매하고 당첨 내역 및 수익률을 알려주는 프로그램

### 구현 기능

---

- 로또 구입 금액 입력 받기

- 몇개 구매했는지 표시하고 번호들 오름차순으로 정렬하여 띄우기

- 당첨 번호 입력 받기

- 보너스 번호 입력 받기

- 당첨 확인 기능

- 수익률 산출 및 표시 기능

### 예외 처리

---

- 구입 금액이 1,000원 단위로 떨어지지 않는 경우

- 당첨 번호가 1~45가 아니고 숫자가 아닌 경우

- 당첨 번호가 6개가 아닌 경우

- 보너스 번호가 1~45 사이의 숫자가 아닌 경우

- 입력이 공백인 경우

- 보너스 번호와 당첨 번호가 겹친 경우
