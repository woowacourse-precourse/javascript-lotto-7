# javascript-lotto-precourse

## 기능 목록 정리

#### 프로젝트에 쓰일 상수 정리
* 로또 번호의 범위
* 발행 번호 개수
* 로또 가격

#### 랜덤을 이용한 발행 번호 뽑는 기능 구현
* 6개의 중복되지 않는 번호

#### 제출 금액에 따라 로또 구매
* 1000원으로 나누어 구입 갯수만큼 출력
* 출력되는 번호는 오름차순

#### 당첨번호 및 지불 금액 입력받는 기능 구현

#### 1 ~ 5등 까지 당첨을 확인하는 기능 구현
* 1 ~ 5등 각 당첨 횟수 출력
* 총 수익 계산 후 수익률 출력

#### 예외사항 처리
