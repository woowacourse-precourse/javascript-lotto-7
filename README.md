# javascript-lotto-precourse

##  💵 로또 발매기 

사용자가 로또 구입 금액과 당첨 번호, 보너스 번호를 입력하면 당첨 내역을 알려주는 로또 발매기입니다. 

<br/>

## 🎯 프로젝트 소개

이 프로젝트는 사용자가 입력한 구입 금액에 따라 로또 번호를 자동으로 발행하고, 입력한 당첨 번호와 비교하여 당첨 결과를 제공합니다. 로또 발매기의 목표는 사용자가 게임을 통해 로또 번호의 조합을 확인하고, 당첨 여부와 수익률을 계산할 수 있도록 돕는 것입니다.

<br/>

## 🔧 주요 기능

1. **로또 발매 기능**  
   - 사용자가 입력한 구입 금액에 따라 로또 번호를 발행합니다. 각 로또 번호는 1부터 45까지의 숫자 중 중복되지 않는 6개 숫자로 구성되며, 오름차순으로 정렬됩니다.

2. **당첨 번호 입력 및 비교**  
   - 사용자는 당첨 번호와 보너스 번호를 입력할 수 있으며, 발행된 로또 번호와 비교하여 당첨 내역을 확인할 수 있습니다.

3. **당첨 내역 및 수익률 계산**  
   - 당첨 번호와 일치하는 로또 번호의 개수를 기준으로 당첨 내역을 표시합니다. 또한, 사용자가 구입한 로또와의 비교를 통해 총 수익률을 계산하여 출력합니다.

4. **예외 처리 기능**  
   - 잘못된 값을 입력했을 때, 에러 메시지를 출력하고 다시 입력을 받도록 처리합니다. 입력된 값이 유효하지 않을 경우 `[ERROR]` 메시지와 함께 에러가 발생합니다.

## ✅ 기능 요구 사항 체크리스트

### 📥 입력

- **구입금액**
  - [X] ‘구입금액을 입력해 주세요.\n’ 입력값을 받는다.
  - [X] 변수 설정: `purchaseAmountInput`

- **당첨 번호**
  - [X] ‘당첨번호를 입력해 주세요.\n’ 입력값을 받는다.
  - [X] 변수 설정: `winningNumbersInput`
  - [X] 쉼표 (,) 기준으로 구분한다.

- **보너스 번호**
  - [X] ‘보너스번호를 입력해 주세요.\n’ 입력값을 받는다.
  - [X] 변수 설정: `bonusNumberInput`

### 📤 출력

1. **구입개수**
   - [X] `{n}개를 구매했습니다.` 라고 출력한다.

2. **발행한 로또 리스트**
   - [X] 구매한 로또 번호를 배열로 보여준다.
   - [X] 로또 한 장에 6개 숫자로 이루어져 있으며, 오름차순으로 정렬된다.
     - 예시: `[8, 21, 23, 41, 42, 43]`

3. **당첨내역**
   - [X] `당첨 통계\n`와 구분선을 출력한다.
   - [X] 일치하는 로또 번호 개수를 기준으로 당첨 횟수를 표시한다:
     - 3개 일치 (5,000원) - {0}개
     - 4개 일치 (50,000원) - {0}개
     - 5개 일치 (1,500,000원) - {0}개
     - 5개 일치 + 보너스 번호 일치 (30,000,000원) - {0}개
     - 6개 일치 (2,000,000,000원) - {0}개

4. **총수익률**
   - [X] `총수익률은 {00.0}%입니다.` 형식으로 출력한다.
   - 구매한 로또와 당첨 번호를 비교하여 총 수익률을 계산하고 로또 게임을 종료한다.

### 🔧 내부 기능

- **구입금액**
  - [X] 로또 1장의 가격은 1,000원으로 고정.

- **로또 번호**
  - [X] 로또 번호의 범위는 1~45까지의 숫자로 제한된다.
  - [X] 로또 한 장당 6개의 중복되지 않는 숫자가 발행된다.

- **당첨 번호**
  - [X] 당첨 번호는 중복되지 않는 6개 숫자와 보너스 번호 1개로 구성된다.
  - [X] 당첨 기준:
     - 1등: 6개 번호 일치 / 2,000,000,000원
     - 2등: 5개 번호 + 보너스 번호 일치 / 30,000,000원
     - 3등: 5개 번호 일치 / 1,500,000원
     - 4등: 4개 번호 일치 / 50,000원
     - 5등: 3개 번호 일치 / 5,000원

- **당첨금액 및 총수익률 계산**
  - [X] 변수 설정: `lottoWinningAmount`, `rateOfReturn`
  - [X] 수익률은 소수점 둘째 자리에서 반올림
    - 계산식: `(당첨 금액 / 구입 금액) * 100`
